# CodeAlpha_Ecommerce_Store

A simple full-stack e-commerce store built with React, Express.js, and MongoDB.

## Features
- Product listing with search and category filter
- Product details page
- Shopping cart
- Order processing / checkout
- User registration and login (JWT authentication)

## Tech Stack
- Frontend: React (Vite)
- Backend: Express.js, Node.js
- Database: MongoDB

## Setup
1. Backend: `cd ecommerce-store && npm install && npm run dev`
2. Frontend: `cd ecommerce-frontend && npm install && npm run dev`

Built as part of the CodeAlpha Full Stack Development Internship.
